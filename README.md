# Store

It is a tool to manage key & values.

## Usage

Add value

```sh
$ store add .path.to value

$ store set .path.to value
```

Get value & keys

```sh
$ store get .path.to

$ store list .path.to
```

Remove value

```sh
$ store rm .path.to
```

## Install

```sh
$ go get github.com/x-color/store
```
